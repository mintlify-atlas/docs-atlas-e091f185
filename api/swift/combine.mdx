---
title: "KMPNativeCoroutinesCombine"
description: "Combine API reference for the KMPNativeCoroutinesCombine package. Wraps Kotlin flows and suspend functions as Combine publishers."
---

The `KMPNativeCoroutinesCombine` package provides [Combine](https://developer.apple.com/documentation/combine) integration for KMP-NativeCoroutines. It converts `NativeFlow` and `NativeSuspend` values into `AnyPublisher`s with full cancellation support.

**Platform requirements:** Swift 5.0, iOS 13.0+, macOS 10.15+, tvOS 13.0+, watchOS 6.0+

<Note>
  All publishers returned by this package are **deferred**. Every new subscription triggers a fresh collection of the flow or a new execution of the suspend function. Store the returned `Cancellable` to prevent immediate cancellation.
</Note>

---

## createPublisher(for:)

Creates an `AnyPublisher` for a `NativeFlow`. Each subscription starts a new, independent flow collection. Disposing the subscription cancels the underlying Kotlin coroutine.

### NativeFlow overload

```swift
public func createPublisher<Output, Failure: Error, Unit>(
    for nativeFlow: @escaping NativeFlow<Output, Failure, Unit>
) -> AnyPublisher<Output, Failure>
```

<ParamField path="nativeFlow" type="NativeFlow<Output, Failure, Unit>" required>
  The native flow to collect. Generated by the Gradle plugin from a `@NativeCoroutines`-annotated Kotlin `Flow` property or function.
</ParamField>

**Returns:** An `AnyPublisher<Output, Failure>` that publishes each value emitted by the flow and completes when the flow finishes, or fails if the flow throws.

```swift
import KMPNativeCoroutinesCombine

let publisher = createPublisher(for: clock.time)

let cancellable = publisher.sink { completion in
    print("Received completion: \(completion)")
} receiveValue: { value in
    print("Received value: \(value)")
}

// Cancel the flow collection
cancellable.cancel()
```

### Unit-emitting flow overload

Use this overload when the flow emits `Unit` values (i.e., `Void` in Swift).

```swift
public func createPublisher<Unit, Failure: Error>(
    for nativeFlow: @escaping NativeFlow<Unit, Failure, Unit>
) -> AnyPublisher<Void, Failure>
```

<ParamField path="nativeFlow" type="NativeFlow<Unit, Failure, Unit>" required>
  The native flow to collect. The flow must emit `Unit` on the Kotlin side.
</ParamField>

**Returns:** An `AnyPublisher<Void, Failure>`.

### NativeSuspend returning NativeFlow overload

Creates an `AnyPublisher` for a suspend function that returns a `NativeFlow`. This is a convenience overload that first awaits the suspend function, then collects the resulting flow.

```swift
public func createPublisher<Output, Failure: Error, Unit>(
    for nativeSuspend: @escaping NativeSuspend<NativeFlow<Output, Failure, Unit>, Failure, Unit>
) -> AnyPublisher<Output, Failure>
```

<ParamField path="nativeSuspend" type="NativeSuspend<NativeFlow<Output, Failure, Unit>, Failure, Unit>" required>
  The native suspend function that returns a `NativeFlow`.
</ParamField>

**Returns:** An `AnyPublisher<Output, Failure>` that publishes the values from the flow returned by the suspend function.

```swift
import KMPNativeCoroutinesCombine

// suspend function that returns a Flow
let publisher = createPublisher(for: randomLettersGenerator.getRandomLettersFlow())

let cancellable = publisher.sink { completion in
    print("Received completion: \(completion)")
} receiveValue: { letters in
    print("Received: \(letters)")
}
```

### Unit-emitting NativeSuspend returning NativeFlow overload

```swift
public func createPublisher<Unit, Failure: Error>(
    for nativeSuspend: @escaping NativeSuspend<NativeFlow<Unit, Failure, Unit>, Failure, Unit>
) -> AnyPublisher<Void, Failure>
```

<ParamField path="nativeSuspend" type="NativeSuspend<NativeFlow<Unit, Failure, Unit>, Failure, Unit>" required>
  The native suspend function that returns a `NativeFlow` of `Unit`.
</ParamField>

**Returns:** An `AnyPublisher<Void, Failure>`.

## createFuture(for:)

Creates an `AnyPublisher` for a `NativeSuspend`. The publisher emits exactly one value and then completes, or fails with an error. Behaviour is equivalent to Combine's `Future`, but backed by the Kotlin coroutine runtime.

### Main overload

```swift
public func createFuture<Result, Failure: Error, Unit>(
    for nativeSuspend: @escaping NativeSuspend<Result, Failure, Unit>
) -> AnyPublisher<Result, Failure>
```

<ParamField path="nativeSuspend" type="NativeSuspend<Result, Failure, Unit>" required>
  The native suspend function to await. Generated by the Gradle plugin from a `@NativeCoroutines`-annotated Kotlin suspend function.
</ParamField>

**Returns:** An `AnyPublisher<Result, Failure>` that either publishes a single value and finishes, or fails with an error.

```swift
import KMPNativeCoroutinesCombine

let future = createFuture(for: randomLettersGenerator.getRandomLetters())

let cancellable = future.sink { completion in
    print("Received completion: \(completion)")
} receiveValue: { value in
    print("Received value: \(value)")
}

// Cancel the underlying coroutine
cancellable.cancel()
```

### Unit-returning overload

```swift
public func createFuture<Unit, Failure: Error>(
    for nativeSuspend: @escaping NativeSuspend<Unit, Failure, Unit>
) -> AnyPublisher<Void, Failure>
```

<ParamField path="nativeSuspend" type="NativeSuspend<Unit, Failure, Unit>" required>
  The native suspend function to await. The function must return `Unit` on the Kotlin side.
</ParamField>

**Returns:** An `AnyPublisher<Void, Failure>`.

### Swift export compat overload

<Warning>
  This overload is deprecated. Migrate away from it once you have fully moved to Swift export.
</Warning>

```swift
@available(*, deprecated, message: "Kotlin Coroutines are supported by Swift export")
public func createFuture<Result>(
    for operation: @escaping () async throws -> Result
) -> AnyPublisher<Result, Error>
```

This overload wraps an arbitrary `async throws` closure in a Combine publisher backed by a Swift `Task`. It exists for source compatibility during migration to Swift export.

## Refining declarations in Swift

You can combine `createPublisher(for:)` with the `@NativeCoroutinesRefined` annotation to expose a Kotlin `Flow` property as an `AnyPublisher` directly in Swift:

```kotlin
// Kotlin
import com.rickclephas.kmp.nativecoroutines.NativeCoroutinesRefined

class Clock {
    @NativeCoroutinesRefined
    val time: StateFlow<Long>
}
```

```swift
// Swift
import KMPNativeCoroutinesCombine

extension Clock {
    var time: AnyPublisher<KotlinLong, Error> {
        createPublisher(for: __time)
    }
}
```

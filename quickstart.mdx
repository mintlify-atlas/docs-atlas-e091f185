This guide assumes you've already completed [installation](/installation). By the end, you'll have a working Kotlin Flow and suspend function callable from Swift with full cancellation support.

## Step 1: Annotate your Kotlin declarations

The KSP plugin generates native wrappers automatically — you just need to add the right annotation.

### Annotating a suspend function

```kotlin
import com.rickclephas.kmp.nativecoroutines.NativeCoroutines

class RandomLettersGenerator {
    @NativeCoroutines
    suspend fun getRandomLetters(): String {
        // Your suspend function implementation
        return "ABC"
    }
}
```

### Annotating a Flow

```kotlin
import com.rickclephas.kmp.nativecoroutines.NativeCoroutines

class Clock {
    @NativeCoroutines
    val time: StateFlow<Long> = MutableStateFlow(0L)
}
```

For `StateFlow` used to track state in a view model, use `@NativeCoroutinesState` instead:

```kotlin
import com.rickclephas.kmp.nativecoroutines.NativeCoroutinesState

class Clock {
    @NativeCoroutinesState
    val time: StateFlow<Long> = MutableStateFlow(0L)
}
```

<Note>
The plugin generates extension properties/functions — it doesn't modify your original class. Build your project after adding annotations to trigger code generation.
</Note>

## Step 2: Call from Swift

Choose your Swift implementation. All three support cancellation.

<Tabs>
  <Tab title="Swift Concurrency (Async/Await)">
    ### Await a suspend function

    ```swift
    import KMPNativeCoroutinesAsync

    let handle = Task {
        do {
            let letters = try await asyncFunction(for: randomLettersGenerator.getRandomLetters())
            print("Got random letters: \(letters)")
        } catch {
            print("Failed with error: \(error)")
        }
    }

    // Cancel the Kotlin coroutine by cancelling the Task
    handle.cancel()
    ```

    ### Collect a Flow as AsyncSequence

    ```swift
    import KMPNativeCoroutinesAsync

    let handle = Task {
        do {
            let sequence = asyncSequence(for: clock.time)
            for try await time in sequence {
                print("Current time: \(time)")
            }
        } catch {
            print("Flow failed with error: \(error)")
        }
    }

    // Cancel the flow collection by cancelling the Task
    handle.cancel()
    ```
  </Tab>
  <Tab title="Combine">
    ### Create a Publisher for a Flow

    ```swift
    import KMPNativeCoroutinesCombine

    let publisher = createPublisher(for: clock.time)

    // Keep a strong reference to the Cancellable
    let cancellable = publisher.sink { completion in
        print("Completed: \(completion)")
    } receiveValue: { time in
        print("Current time: \(time)")
    }

    // Cancel the flow collection
    cancellable.cancel()
    ```

    ### Create a Future for a suspend function

    ```swift
    import KMPNativeCoroutinesCombine

    let future = createFuture(for: randomLettersGenerator.getRandomLetters())

    let cancellable = future.sink { completion in
        print("Completed: \(completion)")
    } receiveValue: { letters in
        print("Got random letters: \(letters)")
    }

    // Cancel the suspend function
    cancellable.cancel()
    ```

    <Warning>
    You **must** keep a strong reference to the `Cancellable`. If it's deallocated, the subscription is cancelled immediately.
    </Warning>
  </Tab>
  <Tab title="RxSwift">
    ### Create an Observable for a Flow

    ```swift
    import KMPNativeCoroutinesRxSwift

    let observable = createObservable(for: clock.time)

    let disposable = observable.subscribe(onNext: { time in
        print("Current time: \(time)")
    }, onError: { error in
        print("Error: \(error)")
    }, onCompleted: {
        print("Completed")
    })

    // Cancel the flow collection
    disposable.dispose()
    ```

    ### Create a Single for a suspend function

    ```swift
    import KMPNativeCoroutinesRxSwift

    let single = createSingle(for: randomLettersGenerator.getRandomLetters())

    let disposable = single.subscribe(onSuccess: { letters in
        print("Got random letters: \(letters)")
    }, onFailure: { error in
        print("Error: \(error)")
    })

    // Cancel the suspend function
    disposable.dispose()
    ```
  </Tab>
</Tabs>
